# wordcount

:whale: count words in a file

**install:**
```
npm i -g countword
```

**use:**
```
echo "Very nice ad-hoc counting." > temp.txt
wordcount temp.txt
5
```

**options:**
- *--all* :whale: count lines, chars
- *--parallel* :whale: use all CPUs
- *--frequencies* :whale: sort by most recent words
